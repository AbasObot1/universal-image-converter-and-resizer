Instruction to use the converter and resizer tool:

  • As Converter and resizer

    • Click on 'Choose file'
    • Choose any image from your device.
    • Set the size as you demand.
    • Select the out image format from the dropdown menu.
    • Click on "Resize and Download" to Save.

  • As Image Converter Only

    • Chose image.
    • Do not touch the sizing. This tool obtains the original size of the image during upload.
    • Select your image format choice suvh as .jpg, .webp, .png, etc.
    • Click on resize and download

    As Image Resizer Only

    • Choose image.
    • Write image size (width(px) and height(px)).
    • Let the image format correspond with the original i.e in .jpg, .png, .WebP, .BMP, etc. by leaving the format at default as set during the upload.
